# Agentic ML Project Template

A GitHub template repository for artifact-first AI-assisted data science, ML, infra, and validation workflows.

## Included
- Root `CLAUDE.md`
- Pre-populated `CONTEXT.md` files
- Core and optional workspaces
- Artifact templates
- Governance logs
- Legacy review workspace
- Bash and PowerShell helper scripts

## Project modes
- **Mode A:** new project from scratch
- **Mode B:** existing repository / legacy enhancement / migration

## Suggested first steps
1. Edit `CLAUDE.md`.
2. Fill `00_brief/problem_statement.md`.
3. Fill `00_brief/success_metrics.md`.
4. Create the first entry in `05_governance/decision_log.md`.
5. Remove optional workspaces you do not need.
6. For existing-repo work, populate `90_legacy_review/` before major changes.

## GitHub template usage
1. Upload these contents to a new GitHub repository.
2. In repository settings, enable **Template repository**.
3. For each new project, click **Use this template**.
4. Clone the new repo locally and customize it.
