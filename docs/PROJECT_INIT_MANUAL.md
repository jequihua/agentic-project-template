# Project Initialization Manual

1. Create a repo from this template.
2. Edit `CLAUDE.md`.
3. Fill `00_brief/`.
4. Document data in `01_data/`.
5. Summarize exploration in `02_analysis/`.
6. Plan and log experiments in `03_experiments/`.
7. Keep `05_governance/` current.
8. For existing repos, fill `90_legacy_review/` before major changes.
