# Prompt Conventions

- coding prompts live in `for_coding_agent/`
- review prompts live in `for_review_agent/`
- filenames use zero-padded sequential numbering
- prompts should point to the exact control artifacts for the milestone
- prompts should include active workspaces, scope, non-goals, and definition of
  done
- prompts should stay narrow enough for a reviewer to verify the result
